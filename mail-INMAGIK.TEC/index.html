INMAGIK
INMAGIK offre servizi legati allo sviluppo di applicazioni web, gestione, processing e visualizzazione dati. La nostra attività va dai servizi di consulenza sulla tecnologia che amiamo e utilizziamo, allo sviluppo di soluzioni software custom e applicazioni e servizi mobile e web based. Ecco i principali strumenti e tecnologie da noi utilizzati:
 
PYTHON
Python è un linguaggio di programmazione ad alto livello, orientato agli oggetti.
 
JavaScript
JavaScript è un linguaggio di scripting lato-client, che viene interpretato dal browser.
 
HTLM5
HTML5 è l'attuale standard per la visualizzazione di documenti web.
 
Django
Django è un framework Python di alto livello per lo sviluppo di applicazioni Web.
 
Celery
Celery è un componente per la gestione dei processi asincroni in Python.
 
AngularJS
AngularJS è un framework JavaScript per lo sviluppo di applicazioni Web client side.
 
D3.js
D3.js è lo standard per la visualizzazioni di dati sul web.
 
Less
Less è un preprocessore che consente di aumentare le potenzialità di CSS.
 
Bootstrap
Twitter Bootstrap è un frontend toolkit HTML, CSS e JavaScript per lo sviluppo web responsive.
 
Apache Cordova
Apache Cordova consente di sviluppare e di compilarle app in HTML5.
 
Ionic
Ionic è un framework HTML5, SASS, AngularJS, per lo sviluppo di app mobile "ibride".
 
PostgreSQL
PostgreSQL è un sistema di database relazionale ad oggetti open source.
 
QGIS
QGIS Desktop open-source per la gestione, visualizzazione, modifica, analisi di dati geografici.
 
PostGIS
PostGIS rende il server PostgreSQL spaziale grazie all'uso di coordinate geografiche.
 
MongoDB
Database NoSQL orientato ai documenti, basato sul formato JSON per la memorizzazione e la rappresentazione dei dati.
 
Linux
GNU/Linux è un valido sistema operativo ed è anche un pilastro dei software 'open source'.
 
Sass
Sass è un'estensione di CSS che aggiunge potenza ed eleganza al linguaggio di base.
