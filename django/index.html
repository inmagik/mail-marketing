CHE COS'E' DJANGO E PERCHE' USARLO?

Django è un framework Python Web di alto livello che incoraggia il rapido sviluppo e il design pulito e pragmatico.


QUINDI, COS'E' DJANGO?

Django è un framework Python, che rende molte cose molto più facili. Utilizzato in moltissimi servizi e siti di successo diffusi su scala mondiale.
L'ORM DI DJANGO

L'ORM è uno strumento molto potente che semplifica l'accesso a diversi sistemi di database mantenendo una base di codice unica. Supporta MySQL, PostgreSQL, Oracle e SQLite.
SICUREZZA RASSICURANTE

Django prende sul serio il problema della sicurezza e aiuta gli sviluppatori a evitare molti errori comuni in questo campo, come SQL injection, cross-site scripting, cross-site request forgery e clickjacking. Il sistema di autenticazione utente di django fornisce un modo sicuro per gestire gli account utente e le password.
ADMIN

Django permette di generare automaticamente una interfaccia di amministrazione del database che può essere utilizzata come backend per la gestione dei contenuti del vostro sito.
PERCHE' USARE DJANGO?

Django si occupa di gran parte dei problemi di sviluppo web, in modo da potersi concentrare sulla scrittura della vostra applicazione. E' gratuito e open source.
CON DJANGO SVILUPPI ALLA VELOCITA' DELLA LUCE

Django è stato progettato per aiutare gli sviluppatori a rendere le applicazioni dall'idea alla realizzazione, nel più breve tempo possibile.
DJANGO E' PYTHON

Python è un framework di alto livello utilizzato da istituzioni e aziende internazionali. Con la sua filosofia "batteries included" e un incredibile ecosistema di framework e librerie, questo linguaggio ha consentito lo sviluppo di molti servizi e applicazioni di successo.
DJANGO FACILITA LE COSE

Django vi facilita le cose! E' un Framework completo con molto a disposizione. Alcuni dei siti più visitati usano la capacità di Django di scalare rapidamente e in modo flessibile per soddisfare le più pesanti richieste di traffico.
